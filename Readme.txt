Se elabora una web sobre una tienda de velas.
Es una web navegable entre páginas de inicio, catálogo, blog, acerca de y contacto.
En el inicio tiene que haber un resumen del resto de páginas por si el usuario no desea navegar (una suerte de página estática).
En el footer se encuentran las redes sociales (es posible que ya existan para una tienda real, desestimar) y un mapa de ubicación (se coloca el obelisco).
El estilo debe transmitir serenidad, elegancia y suavidad.

No utilicé IA.
Sí busqué cómo ordenar algunas cosas, por lo que en el css agregué propiedades como:
- display
- flex-direction
- align-items

Espero que puedan disfrutar de esta primer web.
